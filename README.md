# Installation

## Backend Setup (PHP API)

Please note that the `api` folder must be placed inside the XAMPP `htdocs` directory.

Required structure:

C:\xampp\htdocs\recruitment_platform\api

Example:

After placing the folder:

1. Start Apache and MySQL in XAMPP
2. Open the backend at:
   http://localhost/recruitment_platform/api/

## Frontend Setup

```bash
npm install
npm run dev
